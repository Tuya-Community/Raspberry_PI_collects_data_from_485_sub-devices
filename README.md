
## Overview
Through raspberry PI gateway to collect 485 temperature and humidity sensor, status data of 485 door magnetic switch and control relay action  


## Get started

### Prerequisites

```sh
Python3.0
```

### Compile the code
```sh
PyCharm 2021.2.3
```

### Run the demo
```sh
new_temp_hum_door.py
```
